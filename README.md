# 📈 Stock Trading Platform

A full-stack web application that simulates stock trading with real-time updates and portfolio management.

---

## 🚀 Features

* 📊 View stock prices
* 💰 Buy & sell stocks
* 📈 Track portfolio holdings
* 🔄 Real-time updates
* 📦 Order management system

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
